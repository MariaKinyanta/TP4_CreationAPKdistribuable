          Titre: création d’un APK distribuable
Description : 
Ce projet est une application Android simple avec un bouton dont le texte peut être traduit en anglais ou en français selon la langue du système. L'application utilisera l'éditeur de traduction Android pour gérer les traductions et un APK distribuable sera généré pour une installation facile sur d'autres appareils.

Fonctionnalités:
- Un bouton avec un texte modifiable.
- Traduction du texte du bouton  via l'éditeur de traduction Android.
- Génération d'un APK distribuable pour une installation facile.
- Ajout du TP2

Étapes du projet:
- Lancer le projet Android Studio contenant le TP2
- Générer un APK distribuable pour l'application
- Ajouter un bouton à la mise en page de l'activité
- Définir le texte initial du bouton
- Intégrer l'éditeur de traduction Android dans le projet.

Résultat final:
Une application Android simple avec les informations fournies au TP2 et un bouton dont le texte peut être traduit en anglais ou en francais.
